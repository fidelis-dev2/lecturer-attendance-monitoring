
# Lecturer Attendance Presentation System

Pure front-end simulation built with:
- HTML
- CSS
- JavaScript
- JSON

## Includes
- Login screen
- Dashboard
- Sidebar navigation
- Lecturers page
- Lecture rooms page
- Timetable page
- Attendance monitor
- Reports
- Alerts and notifications
- Settings

## Purpose
This package is for presentation only and shows how the expected live system can look and behave visually.

## Deploy
Open `index.html` directly or upload the folder to GitHub Pages.
